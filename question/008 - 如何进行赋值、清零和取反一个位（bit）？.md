## 问题

在 C/C++ 中，如何进行位操作？

## 最佳回答

#### 位赋值（set）

用`|`操作符来赋值一个位，

```c++
number |= 1UL << n;
```

这段代码是将`number`的第`n`位赋为 1。注意，n 是从 0 开始的。

如果`number`的大小大于`unsigned long`，就需要把`1UL`换成`1ULL`。

#### 位清零（clear）

用`&`操作符来清零一个位，

```c++
number &= ~(1UL << n);
```

将`number`的第`n`位赋为 0。

#### 位取反（toggle）

用`^`操作符来取反一个位（即 0 变 1，1 变 0），

```c++
number ^= 1UL << n;
```

将`number`的第`n`位取反。
