## 问题

```c++
#include <iostream>

int * foo()
{
    int a = 5;
    return &a;
}

int main()
{
    int* p = foo();
    std::cout << *p;
    *p = 8;
    std::cout << *p;
}
```

上面的代码运行不会报错，并且会输出 58，这是为什么？

## 回答

局部变量确实不可以被外部使用，你的这种情况只是因为那块内存区是存在的，但它并不属于 `foo` 函数里的 `a` 所有，所以也不建议你这么用，还是会有奔溃的危险。
