## 问题

一个字符串有很多单词组成，单词间以空格隔开，现在我想遍历这些单词，有什么办法可以高效地实现它呢？

注意，我不想使用 C 的那些字符串操作函数。

下面是我能想到的最好的方案：

```c++
#include <iostream>
#include <sstream>
#include <string>

using namespace std;

int main()
{
    string s = "Somewhere down the road";
    istringstream iss(s);

    do
    {
        string subs;
        iss >> subs;
        cout << "Substring: " << subs << endl;
    } while (iss);
}
```
